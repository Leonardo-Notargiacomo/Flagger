
# Use Case:  As a user, I want to be able to customize the flag


- **Name:** Flag Custmozation
- **Actor:** User
- **Description:** User customizes the flag
- **Pre-condition:** The user must be logged in and allow the usage of their location.


## Main Scenario:

1. User makes an attempt to mark change the flag design
2. System displays the customization window 
3. User changes the flag setting
4. System displays the map with the updated design


## Result:
The flag design is updated.
