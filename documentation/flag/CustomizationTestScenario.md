# Test Scenarios: Customize the flag

## Test Scenario 1: The user does not cancel the changes

**Preconditions**:  
- The user is logged in.  
- The user allowed the usage of their location.

**Steps**:  
1. User attempts to change the flag design  
2. The system displays the customization window
3. User selects new design and saves
4. System displays the map.  


**Expected Result**:  
The flag design is updated
System saves the new flag design and displays all the flags with the new design.  

---

## Test Scenario 2: The user cancels the changes


**Preconditions**:  
- The user is logged in.  
- The user allowed the usage of their location.

**Steps**:  
1. User attempts to change the flag design  
2. The system displays the customization window
3. User selects new design and cancels
4. System displays the map 


**Expected Result**:  
The flag design remains unchanged
