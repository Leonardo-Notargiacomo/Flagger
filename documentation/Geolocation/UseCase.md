
# Use Case: As a user, I want to view my current location on a map so that I can understand where I am in the app world.



- **Name:** Home page 
- **Actor:** User
- **Description:** User enters the home page, where the map, geolocation and the navigation bar are displayed
- **Pre-condition:** The user must be logged in.

## Main Scenario:

1. System displays the pop up window to ask for the user`s geolocation permisssion 
2. User accepts the request.
3. System displays the home page.

## Result:

- Home Page is displayed 
## Exceptions
### 1 User does not accept the request 
1. Geolocation is not displayed
