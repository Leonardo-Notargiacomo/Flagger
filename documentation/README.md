# PRJ3 Documentation

This Directory concerns itself with the creation of the Group app **'GO'**.

## Application Idea

The App idea is to create an App that allows one to mark places of significance on the map, as a travel book. In addition to **flagging** these places, the User can upload a picture to each place.
It works as gamification; in that regard, flagging works similarly to manually placing a Flag at the place of interest. Imagine putting the American Flag on the moon for a more hyperbolical example. 
Places of interest include parks, sights, and other places worth visiting. Places like supermarkets, malls, and hospitals do not normally count as places of interest; this, of course, is debatable. Examples include the **Hospital de Sant Pau** in Barcelona and the **Galeries Lafayette** in Paris.
As a central part of the gamification, badges are included in the game, for example, for visiting five places.
To contribute to the gamification, the ability to add friends and compete with them is of interest.

## Necessities

* **Camera** - Camera is used to upload a Profile Picture, and add it to the map as remembrance.
* **User-Location** - To show the User's location on the map. Further used to check whether a User's proximity is close enough to a place to flag it.
* **Map** - Visualisation for the User's location and possible points of interest.
* **Push-Notification** -  Duolingo style rememberer to ensure that the User continues exploring the world with us.
