# Design Choices

**Collective Designs**

The Design is based upon the [Retro Map Design](https://snazzymaps.com/style/118034/retro-by-google). 

Main colour: #ebe3cd — this is used for the base geometry of pretty much all features.

Secondary colour: #523735 — used for labels’ text fill, providing contrast.

Detail / accent colour: #e98d58 — used for accent and further detailed exercise.
